- 程序 py-demo.py是主程序，主要是3d检测，增加了鲁棒性，可在没有足够空间信息时退化为2d。
- generator.py用于生成简单的测试视频，需要更多请自行咨询AI。
- 在测试结果中，仅生成视频和尾号为47的视频满足了需求，其他均存在一些问题，可能是寻找的测试视频原因，比如发球不规范（按照测试要求是向上发球）和球超出视频范围。
